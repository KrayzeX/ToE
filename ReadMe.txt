При вводе контурной матрицы контурные токи нужно задавать так, чтобы, если у контуров есть
общие реактивные элементы, направления контурных токов на них совпадало.
В контуре с катушкой все сопротивления в контурной матрице задавать с минусом.

Если в цепи конденсатор и катушка, то конденсатор будет задаваться [1 0 0] в матрице ИН
на строке с номером, соответствующим ветви конденсатора. Для катушки аналогично
 - [0 1 0] в матрице ИТ. В списке параметров ReactValue1 - ёмкость конденсатора.

Если в цепи два конденсатора, то конденсатор с меньшим номером ветви задаётся
[1 0 0] в матрице ИН на строке с номером, соответствующим ветви конденсатора,
а с большим - [0 1 0] в той же матрице.
В списке параметров ReactValue1 - ёмкость первого конденсатора.

Аналогично для цепи с двумя катушками.
